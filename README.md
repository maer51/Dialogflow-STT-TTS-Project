# Dialogflow-STT-TTS-Project
Google Dialogflow와 STT, TTS를 android에 연동한 앱

## 사용한 도구 
* Android Studio (Java)
* Google Dialogflow
* Google TTS API 
 (안드로이드 안에 무료로 TTS 내장API를 지원한다. 삼성핸드폰은 samsung TTS API를 지원한다.)
